# Utilix Frontend

Next.js frontend for Utilix, a collection of useful online tools.

## Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Scripts

```bash
pnpm lint
pnpm build
pnpm start
```

## Assets

Image assets live under `public/images`.
